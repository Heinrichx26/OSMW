# Raw Data Package

No private raw data package is required.

The numerical inputs are generated by the scripts in `src/` using fixed random
seeds. The generated CSV and JSON tables are included in `data/` and `results/`
so that figures can be reproduced without rerunning every simulation.

The public benchmark component uses Munich Quantum Toolkit Bench circuits
retrieved through the `mqt.bench` package. Those source circuits are public
benchmark definitions and are not vendored here. To regenerate the public
benchmark rows, install the optional dependencies listed in `requirements.txt`
and run:

```bash
python src/r41_benchmark_validation.py
python src/r41_fixed_error_budget.py
```

The decision-validation tables, analytic same-support separation, and random
same-support coefficient validation are regenerated by:

```bash
python src/r41_decision_validations.py
python src/r41_same_support_separation.py
python src/r41_weighted_observable_validation.py
```

For a local quick validation that avoids public benchmark downloads, run:

```bash
python src/reproduce_all.py --quick --skip-mqt
```
